# Sidebar Keywords / Figures Vertical Tabs

**Goal:** Split the left browse panel into vertical tabs (Keywords | Figures) with an adaptive occurrences table and figure validation powered by the validation endpoint.

**Approach:**
1. Store `pdfPath` on Project so validation can re-render page images.
2. `POST /api/projects/:id/figures/:pageNumber/validate` — builds extracted concepts from DB callouts, runs Gemini validation.
3. Frontend: vertical tab strip right of sidebar list; Figures tab lists only illustrations with `figureNumber`; main table adapts to keyword or figure selection; anomalies panel for validation results.
