# Bundled Figure Discovery in Validate-All

**Goal:** Run figure discovery and referential-integrity validation from the Figures tab without requiring keyword Extract first.

**Approach:**
1. `validate-all` scans PDF pages with illustrations (not DB-first).
2. Per-page LLM mode: `withConcepts` (after Extract), `standalone` (figureNumber only), or `discoverAndValidate` (combined, fresh pages).
3. Upsert `Illustration` on `(projectId, pageNumber)` — fill `figureNumber` if null only; never write callouts/keywords.
4. Frontend: Validate enabled when PDF linked; refresh figures list after validate-all.
