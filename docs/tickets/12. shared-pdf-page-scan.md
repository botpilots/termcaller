# Shared PDF Page Scan Orchestrator

**Goal:** DRY up Extract and Validate page iteration — single PDF session, shared concurrency constant, reusable illustration-page scan.

**Approach:**
1. `PdfSession` caches parsed document; `extractPageData` delegates to it.
2. `scanPdfPages()` orchestrates concurrent page iteration with adjacent-image helper.
3. Extract and Validate use shared `PDF_PAGE_CONCURRENCY`.
