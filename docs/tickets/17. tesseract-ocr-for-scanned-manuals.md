# Tesseract OCR for Scanned Manuals

**Goal:** Extract text from image-only PDFs (e.g. `551827.pdf`, Nissan R33) for the service-manual corpus.

**Approach:** Use open-source Tesseract + `pdftoppm` (poppler). Output `{basename}_hocr_searchtext.txt` sidecars. Corpus build uses normal PDF extraction first; on low chars/page yield it logs a warning pointing to `npm run ocr:pdf`, and uses an existing sidecar if present. Incremental rebuild skips unchanged sources via SHA-256 checksums in `build-manifest.json`.
