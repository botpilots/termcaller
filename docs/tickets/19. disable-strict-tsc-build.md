# Disable Strict TypeScript Checks at Build

**Goal:** Remove blocking TypeScript type checks (`tsc`) from the build process and disable strict mode to allow deployments to proceed even with non-critical type errors.

**Context:** During hackathon development, strict type checking can block rapid deployments to Cloud Run. The user requested removing these checks at build time.

---

## Changes

1.  **Root `package.json`**: Remove `&& npx tsc` from the `build` script.
2.  **Frontend `package.json`**: Remove `tsc -b && ` from the `build` script.
3.  **Backend `tsconfig.json`**: Set `"strict": false` and disable other strict options like `noUncheckedIndexedAccess` and `exactOptionalPropertyTypes`.
4.  **Frontend `tsconfig.app.json`**: Ensure strict mode is disabled (or not enabled).
