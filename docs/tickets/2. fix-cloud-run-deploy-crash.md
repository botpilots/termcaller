# Fix Cloud Run Deploy Crash

**Goal:** Fix container startup crash on Cloud Run caused by invalid Express SPA fallback route syntax.

**Context:** Deploy revision `termcaller-dev-00006` crashed with `PathError: Missing parameter name at index 2: /*` from path-to-regexp v8 (Express 5). Frontend Docker build also failed on TypeScript errors.

---

## Changes

1. Replace `app.get('*'|'/*')` catch-all with middleware that skips `/api` paths — works on Express 4 and 5.
2. Fix `AuthContext.tsx` type-only import for `ReactNode` (verbatimModuleSyntax).
