# Disable Gemini Thinking for Extraction

**Goal:** Speed up straightforward callout extraction on dense illustration pages (e.g. BioDrill page 26) by limiting model thinking.

**Root cause:** `gemini-3-flash-preview` defaults to automatic thinking budget. Visual callout scanning is exploration, not deep reasoning — thinking tokens add latency and contribute to 60s timeouts under concurrent load.

**Approach:** Use Gemini 3 `thinkingConfig.thinkingLevel: LOW` on extraction (not legacy `thinkingBudget`).

**Status:** IMPLEMENTED — updated to `LOW` after `thinkingBudget: 0` trial.
