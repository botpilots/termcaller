# Delete project

**Goal:** Allow users to delete the selected project (and its PDF) from the dashboard header.

**Why:** Projects accumulate during testing; users need a way to remove a project without manual DB cleanup.

---

## Scope

- `DELETE /api/projects/:id` — auth + ownership, cascade DB delete, remove PDF from disk, close SSE clients.
- Header delete button for the selected project, with confirmation.
