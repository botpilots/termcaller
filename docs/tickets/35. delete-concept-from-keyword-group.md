# Delete concept from keyword group

**Goal:** Allow removing a concept (figure occurrence) from a keyword group via a delete button next to the concept title in the editor.

**Why:** Extracted concepts are often wrong or duplicates; users need to remove them without re-running extraction.

---

## Scope

- `DELETE /api/keywords/:id/occurrences` — same occurrence identity as PATCH (page, figure, identifiers)
- `deleteOccurrence` in `occurrenceEditService` — delete callouts, disconnect concept from keyword when no occurrences remain, delete empty keyword/concept
- Delete button to the right of the concept title in keyword `OccurrenceForm`
