# Ignore occurrence (soft exclude from TBX)

**Goal:** Replace Delete with Ignore; keep data but exclude from TBX export.

**Status:** IMPLEMENTED

---

## Scope

- `Concept.excludedFromExport` boolean (default false)
- `POST /api/keywords/:id/occurrences/ignore` replaces hard DELETE
- TBX export skips excluded concepts
- UI: Ignore button + modal copy
