# Group Duplicate Callouts in Gemini Schema

**Goal:** Reduce LLM repetition when many callouts on a page refer to the same concept (e.g. multiple "seed hose" callouts on page 31).

**Approach:** Change `calloutIdentifier` from a single string to `calloutIdentifiers: string[]`. One extracted concept can list all identifiers that share the same `sourceTerm` and `functionalDescription`. Processing fans out into one DB callout per identifier.
