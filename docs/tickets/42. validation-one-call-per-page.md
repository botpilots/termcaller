# Validation: one LLM call per page

**Goal:** Stop duplicate cross-figure validation warnings by making exactly one Gemini call per page (figures[] response), scoped per illustration. Filter non-callout labels (serial numbers, part numbers).

**Root cause:** `validateAllProjectFigures` called Gemini once per DB illustration on the same page image, so each call saw all page callouts and reported other figures' labels as unreferenced.

**Approach:** Mirror extraction's ordered `figures[]` pattern — one call returns per-figure validation; map results to DB figure numbers by reading order. Filter label mismatches against page text so textIdentifier must appear next to the part name.
