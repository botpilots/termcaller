# Persist validation results on Illustration

**Goal:** Store referential-integrity validation results in the database so they survive page reload and project re-open.

**Root cause:** `validateAllProjectFigures` returns validation in the API response but never writes it to Prisma. Only illustration rows are upserted on discovery; existing figures get in-memory results on the frontend only.

**Approach:** Add `validationResult` (JSON) and `validatedAt` on `Illustration`. Save after batch and single-figure validation. Hydrate frontend `figureValidations` from `GET /figures`.
