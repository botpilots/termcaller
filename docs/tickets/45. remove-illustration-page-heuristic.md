# Remove illustration page heuristic

**Goal:** Stop pre-filtering PDF pages by vector/raster operator counts. Send every page to Gemini extraction and validation; let the model return empty results for text-only pages.

**Motivation:** The `pageHasIllustrations` heuristic (constructPath + setStrokeRGBColor thresholds) skips most pages in EPS-origin manuals such as Planmed Sophie, even when they contain labeled exploded views.

**Approach:**
1. Remove `hasIllustrations` from `ParsedPage` and delete `pageHasIllustrations`.
2. Remove pre-Gemini skip checks in processing, validation, extraction collector, and dev scripts.
3. Remove `illustrations-only` filter from `scanPdfPages`.
