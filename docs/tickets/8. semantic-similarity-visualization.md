# Semantic Similarity Visualization

**Goal:** Analyse concept definition similarity per keyword using Gemini embeddings and visualize clusters with a centroid-based circle layout.

**Context/Tech Stack:** `gemini-embedding-001`, Express, React Dashboard with toggleable Similarity tab.

---

## 1. Overall Goal & Scope

Backend endpoint `/api/keywords/:id/analyze-similarity` embeds missing concept definitions, computes a centroid, scores cosine similarity, and flags outliers (< 0.85). Frontend adds a toggleable Similarity tab with an analyse button; concepts render as circles positioned by distance from a central centroid circle.

## 2. PR Implementation Plan

- **PR 1:** Embedding service, vector math, keywords route
- **PR 2:** Similarity tab + cluster visualization component
